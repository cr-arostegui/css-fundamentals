Position Property

Static:

static block content: This is the default value, elements with this position render in order.

Absolute:

Example 1 - When the first parent is the browser window.

absolute block content 1: The element is positioned relative to its first positioned (not static) ancestor element.

Example 2 - When we set the container as parent (position: relative).

absolute block content 2: The element is positioned relative to its first positioned (not static) ancestor element.

Sticky:

sticky block content: the element is positioned based on the user's scroll position. (Take into account browser version to support this feature)

Relative:

relative block content: the element is positioned to its normal position.

Fixed:

fixed block content: the element is positioned relative to the browser window.
